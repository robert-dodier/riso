Robert Dodier: The RISO Project

``Think locally, compute globally''

Welcome to the RISO project web page. Here you'll find source code, documentation, and other resources related to RISO, an implementation of distributed belief networks in Java.

The RISO source code is released under the GNU Public License. You can download the RISO source as a ZIP file.

Technical papers on aspects of the RISO project can be found on my publications page.

My Dissertation.

The current draft of my dissertation, "Unified prediction and diagnosis in engineering systems by means of distributed belief networks," is available for download as PostScript and PDF.

I have separated it into sections as listed here. The PDF files are somewhat smaller than the PostScript. I've noticed that Acrobat Reader doesn't display some mathematical symbols until you zoom in, so if something looks strange, try zooming in.

  • Entire dissertation. (PostScript) (PDF)

  • Chapter 2, "Introduction to graphical probability models." (PostScript) (PDF)

  • Chapter 4, "Overview of the RISO belief network system." (PostScript) (PDF)

Example RISO belief networks.

My dissertation refers to these belief networks.

  • Chapter 1.
    • combine-reports.riso, combines reports from remote RISO sites.
    • portland-report.riso, model of observations of Elvis in Portland.
    • cambridge-report.riso, model of observations of Elvis in Cambridge.
    • lampertheim-report.riso, model of observations of Elvis in Lampertheim.
    • istanbul-report.riso, model of observations of Elvis in Istanbul.
  • Chapter 7.
    • electrical-demand.riso, model of electrical energy use in a building.
    • heat-transfer-consts.riso, a couple of constants referenced by electrical-demand.riso, implemented as belief network variables.
    • generate-daily-max-bn.awk, an AWK program to generate a belief network to compute a distribution over daily maxima for a given day of the year. Here is a typical output, for day 200.
    • generate-monthly-max-bn.awk, an AWK program to generate a belief network to compute a distribution over monthly maxima for a month of the year. Here is a typical output, for July.
  • Chapter 8.
    • Tdblvg.riso, a belief network model of part of a refrigeration system.
    • damper-tbn.riso, a temporal belief network model of a mixing box damper.
